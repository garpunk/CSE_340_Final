Introduction:

what this site is.

This site is designed as kindof an auxillary site to a college campus tech website that allows for users to submit tickets for requests of things they need fixed, specifically students and faculty. It's basically a helpdesk for the IT department of a school campus.

pgAdmin ERD

![ERD for the site](erd_it_site.png)


Explanation of User roles:

User - can create, update, and delete their own support tickets. can view their support tickets and comments on those support tickets, can comment on their support tickets. 

Staff - has the same abilities as the user, but can also access the ticket log and view all support tickets; can assign tickets to themselves and update the status of those tickets and leave comments on all tickets. 

Admin - has the same abilities as the two previous roles, but can also manage users and assign users to different roles, can also manage ticket categories and can delete categories and reassign tickets depending on the category.
